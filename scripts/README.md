# Скрипты для управления играми

## create-games-db.sh

**Надежный скрипт для создания завершённых игр со счётами напрямую в БД.**

### Возможности
- ✅ Создаёт 3 игры (Четверка, Восьмёрка, Десятка)
- ✅ Регистрирует нужное количество игроков
- ✅ Создаёт матчи со случайными очками (0-24 за матч)
- ✅ Обновляет счётчик игр для каждого игрока
- ✅ Закрывает старые неполные игры перед созданием новых

### Запуск
```bash
bash scripts/create-games-db.sh
```

### Параметры
Скрипт автоматически:
- Загружает список всех игроков из API
- Использует текущую дату для всех событий
- Начинает игры в 18:00, заканчивает в 20:00
- Устанавливает формат AMERICANA с режимом ROUND_ROBIN
- Проставляет режим подсчёта очков (POINTS, 6 за матч)

### Результаты
- 3 завершённые игры (FINISHED status)
- 5-6 матчей с реальными очками
- Рейтинги игроков обновлены
- Таблица лидеров заполнена

### Почему этот способ лучше API?
1. **Надежность** - не зависит от авторизации API
2. **Скорость** - прямое взаимодействие с БД
3. **Полнота** - гарантирует корректное состояние БД
4. **Воспроизводимость** - всегда работает одинаково

### Таблица соответствия команд API ↔ БД

| Операция | API | БД |
|---|---|---|
| Создание события | POST /api/events | INSERT INTO events |
| Регистрация игроков | POST /events/{id}/register | INSERT INTO registrations |
| Создание раунда | Автоматически | INSERT INTO rounds |
| Создание матчей | Автоматически | INSERT INTO matches |
| Проставка очков | POST /matches/{id}/score | INSERT INTO match_draft_scores |
| Обновление рейтинга | POST /events/{id}/finish | UPDATE players.games_played |
